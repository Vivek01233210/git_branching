git branch -  to check available branches
git branch a - creates branch with name 'a' in local
git remote add origin 'ssh_address of that branch' - for linking local branch with
new branch created in github.
git checkout a - to switch to branch 'a' in local
git merge a - to merge branch 'a' with master branch
git push -u origin master - to update the merging in github as well.
git branch -d a - to delete branch 'a' from local.
git push origin --delete a - to delete branch 'a' from github.
